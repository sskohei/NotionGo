## About
このレポジトリはGoで作成したNotionのデータベースをCLIで操作するツールのレポジトリです。

## 概要
NotionGoはすでに作成してあるNotionのデータベースのdata_source_idとNotionApiKeyを入力することによりNotionのデータベースを表として出力、データの追加をすることができるツールになっています。
